Changelog
=========


v0.1.0
------

- First release.
- Added classes for building some of the tags:

  - <a></a>
  - <abbr></abbr>
  - <acronym></acronym>
  - <address></address>
  - <area>
  - <b></b>
  - <base>
  - <button></button>
  - <canvas></canvas>
  - <caption></caption>
  - <cite></cite>

Caveats:

- Korona has no ability to construct inner tags. (For ex. <address><p>Hi There</p></address>)
- Korona will be added with constructing the inner tags in the next release.
